NeckGo 是一款专为"久坐低头族"打造的游戏化颈椎健康 App，支持 iOS 和 Android 双平台。它的核心思路很有意思——**不靠说教，靠游戏**。

## 三个核心玩法，一步步"骗"你动起来

### 🎮 户外日记

戴上 AirPods（或开启手机摄像头），左右侧屈脖子，你就可以控制角色骑行、冲浪、滑雪、划船……一路收集能量道具，同时完成了颈椎侧屈拉伸。

玩法逻辑和跑酷游戏差不多，只不过不是用手指划屏幕，是用你那僵硬的脖子在操控。你说这不是"骗你运动"是什么？

### ✈️ 飞行之旅（仅 iOS + Apple Watch）

连上 Apple Watch，摆动手臂，控制角色翱翔各地，从北京飞到巴黎，从东京飞到纽约，解锁全球城市徽章。相当于在屏幕里环游世界，同时手臂和颈椎也跟着动了。

### 🐦 坐姿守护

戴上耳机或开启摄像头后台运行，一旦检测到你低头超标，就会发出百谷鸟鸣声来提醒你。保持良好坐姿一段时间，还能获得果实奖励，积累成就。

这个功能特别适合对着电脑工作的人，就是那种"工作的时候不知不觉就开始缩着脖子"的状态，让 NeckGo 在旁边盯着你。
